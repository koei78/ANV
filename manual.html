ANV 請求・稼働管理システム操作マニュアル

本マニュアルでは ANV 請求・稼働管理システムの各機能の使い方を説明します。

ブラウザの「印刷」→「PDFとして保存」でPDF化できます。

1. ログイン

  1. ブラウザで表示されたURLにアクセスします。
  2. 「ユーザー名」と「パスワード」を入力して「ログイン」ボタンをクリックします。
  3. ログインに成功するとダッシュボードが表示されます。

2. ダッシュボード

  1. 今月の請求額（月初・月末）・粗利・稼働件数を KPI カードで確認できます。
  2. 担当営業別の売上・粗利が表とグラフで確認できます。
  3. 直近6ヶ月の粗利推移グラフが表示されます。
  4. 未対応・振込待ちの件数がアラートとして表示されます。

3. 稼働登録

  1. サイドバーの「稼働登録」をクリックします。
  2. 対象年・月、クライアント先、稼働者を選択します。
  3. クライアントを選択すると稼働店舗が自動入力されます。
  4. 出勤数（0.5日単位可）と交通費を入力します。
  5. 「登録」ボタンをクリックして保存します。

4. 稼働一覧・ステータス管理

  1. サイドバーの「稼働一覧」をクリックします。
  2. 月・クライアント・ステータスで絞り込みができます。
  3. 「次のステータスへ」ボタンをクリックしてステータスを進めます。
  4. ステータスの流れ：未対応 → 月初請求書送付済 → 月末請求書送付済 → 振込待ち → 振込完了

5. 振込一覧

  1. サイドバーの「振込一覧」をクリックします。
  2. パートナー企業への振込が必要な稼働記録を月別に確認できます。
  3. パートナー企業・稼働者・振込金額が一覧表示されます。
  4. 振込完了後はステータスを「振込完了」に更新してください。

6. 請求書管理

  1. サイドバーの「請求書管理」をクリックします。
  2. 「新規作成」ボタンから請求書を作成します。
  3. クライアント・対象年月を選択すると稼働実績が自動プレビューされます。
  4. 発行日・支払期限・備考を入力して「作成」をクリックします。
  5. 請求書一覧から「PDF」ボタンをクリックしてPDFをダウンロードできます。
  6. ステータスは「下書き → 発行済 → 入金済」の順に管理します。

7. 担当営業レポート

  1. サイドバーの「担当営業レポート」をクリックします。
  2. 対象月を選択（または「全期間」）して絞り込みます。
  3. 担当営業ごとの売上・パートナー支払・粗利・粗利率を確認できます。
  4. 棒グラフで各担当の売上と粗利を視覚的に比較できます。
  5. 詳細セクションでは担当ごとのクライアント別内訳も確認できます。

8. マスター管理 — クライアント先

  1. サイドバーの「クライアント先」をクリックします（管理者のみ編集可）。
  2. 「新規登録」ボタンから新しいクライアントを追加します。
  3. 会社名・稼働店舗名・担当者名・メールアドレスを入力します。
  4. 受け単価（日当）・支払いサイト・請求書送付方法を設定します。
  5. 担当営業をドロップダウンから選択します。

9. マスター管理 — 担当営業

  1. サイドバーの「担当営業」をクリックします（管理者のみ編集可）。
  2. 「新規登録」ボタンから担当営業を追加します。
  3. 氏名を入力して「登録」をクリックします。
  4. 登録した担当営業はクライアント先の編集画面で選択できます。

10. マスター管理 — パートナー企業・稼働者

  1. 「パートナー企業」ではパートナー会社の情報・振込先口座を登録します。
  2. 「稼働者」では稼働者の氏名・所属パートナー企業・パートナー単価を登録します。
  3. 稼働者は稼働登録時にドロップダウンで選択します。

11. 会社設定

  1. サイドバー下部の「会社設定」をクリックします（管理者のみ）。
  2. 請求書に表示される会社名・住所・電話番号を設定します。
  3. 振込先の銀行情報（銀行名・支店名・口座種別・口座番号・口座名義）を入力します。
  4. 請求書の備考欄に表示するテキストを設定できます。
